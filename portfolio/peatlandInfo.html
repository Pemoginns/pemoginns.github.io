Irish Peatland Project

Introduction

This was my final year project for my bachelor's degree in university. I investigated the maintenance of peatlands in Ireland today and wrote my thesis on this subject. I created this map using Mapbox and Leaflet.js. The map itself holds more than 50 different sites around Ireland.

The map is dominated by Bord Na MÃ³na project sites, which are most common in the center of Ireland. My research provided this interactive map that displays information on any project site the user clicks on.

For this project, I referenced websites like Geohive, a platform presenting statistical data in a map format. Inspired by their filtering features, I implemented similar functionality in my application. The suggested sites section highlights the user's choice on the map, making it easier to find a specific site. Once a site is chosen, the website will open a new section at the bottom with contact information or a display of the company's website for quick access.

Example

  • We pick a site on the map and click open.

  • After some JavaScript processing, the website of the site is displayed below the map (auto-scroll is on).
