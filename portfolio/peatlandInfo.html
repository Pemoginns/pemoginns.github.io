Irish Peatland Project

Introduction

This was my final year project for my bachelor's degree in university. I investigated what was being done about the maintenance of peatlands in Ireland and wrote my thesis on this subject. I created this map using a JavaScript library.

The map is dominated by Bord Na MÃ³na project sites, which are most common in the center of Ireland. My research provided this interactive map that displays information on any project site the user clicks on.

For this project, I referenced websites like Geohive, which presents statistical data in map format. Inspired by their filtering features, I implemented similar functionality in my application.

Example

  • We pick a site on the map and click open.
  • After some JavaScript processing, the website of the site is displayed below the map (auto-scroll is on).
