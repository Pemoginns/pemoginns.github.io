Irish Peatland Project

Introduction

This was my final year project for my bachalors degree in university. I had to investigate what was being done about the maintenance of peatlands in Ireland. I wrote my thesis about this. I created this map using an javascript library. The map is mostly dominated by BordNaMona project sites. These are most common in the around the center of Ireland. My research provided this interactive map which held and could display infomation on any project site the user clicked on.

For this project, I regarded websites like Geohive. I found this website to be useful as it presents statistic data in map format. I liked the idea of filtered based on a company and so I implemented this into my application.

Example

  • We pick a site on the map and click open
  • After some javascript stuff happens, the website of the site is displayed below the map(auto scroll is on)
