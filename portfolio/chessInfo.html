Puzzle Chess

This was another project I created for college. I designed a chess game using Python and Pygame. It worked in a technical sense, but I’ve wanted to port it to Unity for better performance. The game is based on a series of numbers and letters organized into what’s called **FEN notation**. You can add as many different FENs as you want to the game, allowing you to play out any scenario you like.

There’s also a random setting in the code that generates random games. While not fully endless, it can run for a while and is great for killing time!
