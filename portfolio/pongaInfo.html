Introduction

This was a game I developed in my 2nd year of university using Python with the Pygame framework.
The aim of the game is simpleâ€”it's designed to waste time. In an ideal world, I would have fully developed this game.
The concept of the game is essentially Pong but with a twist. The player (square) has to try to bounce the ball into the green square
before the time runs out. Every round gets quicker-paced, and occasionally a rectangle or polymorph would spawn.
After the player achieves the goal, the game restarts with increased difficulty.
