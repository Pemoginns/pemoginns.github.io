# Portfolio Website UI Kit

## Overview
A component-level recreation of Peter McGuinness's personal portfolio website at pemoginns.github.io.

## Design System
- **Framework:** Tailwind CSS + Font Awesome 6
- **Primary Font:** System sans-serif
- **Accent Font:** Montserrat (sub-pages)
- **Palette:** Ireland-inspired greens (#006837 → #169b62)
- **Background:** Animated diagonal gradient
- **Cards:** White, rounded-xl, shadow-lg, hover lift

## Components
- `Header.jsx` — Sticky nav with name + links + social icons
- `HeroSection.jsx` — About / intro section
- `ProjectCard.jsx` — Project card with hover-reveal details
- `SkillsSection.jsx` — 3-column tech skills grid
- `ContactSection.jsx` — Info + contact form
- `Footer.jsx` — Social links + copyright
- `SkillBadge.jsx` — Pill tech tag
- `SubPageLayout.jsx` — Dark-mode inner page layout

## Screens
1. **Home** — Full portfolio page (default view)
2. **Project Detail** — SmartStep dark sub-page

## Usage
Open `index.html` to see the interactive portfolio prototype.
